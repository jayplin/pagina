
# Soft and Messy

Webapp to access posters of soft and messy magazine in a digital way.  


## Installation


```bash
  pnpm install
  ```
    
## Run Locally

Generate static assets

```bash
  pnpm build
```

Development server 

```bash
  pnpm dev
```


## Deployment

Hosted on Netlify. Deployment is triggered on push.
